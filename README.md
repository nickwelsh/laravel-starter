My starter kit for [Laravel 12](https://laravel.com/), [Inertia 2](https://inertiajs.com/), and [React 19](https://react.dev/).

## Getting Started

1. Clone the repository
2. Run `composer install`
3. Run `bun install`
4. Run `php artisan dev:setup` to set up env variables and database
5. Run `composer dev` to start the development server
